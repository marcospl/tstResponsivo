Guia de Referência de Tags Condicionais do WordPress

comments_open()

Esta tag condicional verifica se os comentários são permitidos no post.

<?php comments_open( $post_id ); ?>

Parâmetros

$post_id -> O ID do Post

Parametro obrigatório

Padrão: None

is_404()

Esta tag condicional verifica se a página de erro 404 está sendo exibida

<?php is_404(); ?>

Parâmetros

Esta tag não requer parâmetro

is_admin()

Esta tag condicional verifica se a página de adminstração está sendo exibida

<?php is_admin(); ?>

Parâmetros

Esta tag não requer parâmetro

is_archive()

Esta tag condicional verifica se qualquer tipo de página de Arquivo está sendo exibida, seja ela de: Autor, Categoria, Tags, ect 

<?php is_archive(); ?>

Parâmetros

Esta tag não requer parâmetro

is_attachment()

Esta tag condicional verifica se qualquer tipo de anexo está sendo exibido

<?php is_attachment(); ?>

Parâmetros

Esta tag não requer parâmetro

is_author()

Esta tag condicional verifica se uma página de arquivo de autor está sendo exibida

<?php is_author($author); ?>

Parâmetros

$author -> O ID do autor ou Nickname do autor

Parametro opicional

Padrão: None

is_category()

Esta tag condicional verifica se uma página de arquivo de categoria está sendo exibida

<?php is_category($category); ?>

Parâmetros

$author -> O ID da categoria ou o título da categoria ou o slug da categoria

Parametro opicional

Padrão: None

is_comments_popup()

Esta tag condicional verifica comentários em janela popup

<?php is_comments_popup(); ?>

Parâmetros

Esta tag não requer parâmetro

is_date()

Esta tag condicional testa para ver se a página é uma página de arquivo data base

<?php is_date(); ?>

Parâmetros

Esta tag não requer parâmetro

is_day()

Esta tag condicional testa para ver se a página é uma página de arquivo da data base

<?php is_day(); ?>

Parâmetros

Esta tag não requer parâmetro

is_feed()

Esta tag condicional verifica a consulta para o feed

<?php is_feed(); ?>

Parâmetros

Esta tag não requer parâmetro

is_front_page()

Esta tag condicional verifica se a página inical está sendo exibida (usado para quando for páginas estáticas)

<?php is_front_page(); ?>

Parâmetros

Esta tag não requer parâmetro

is_home()

Esta tag condicional verifica se a página inical está sendo exibida (geralmente usada quando a página home.php está inclusa no tema)

<?php is_home(); ?>

Parâmetros

Esta tag não requer parâmetro

is_local_attachment()

Esta tag condicional verifica se o anexo é uma URL local e se é realmente um anexo

<?php is_local_attachment($url); ?>

Parâmetros

$url -> URL para consulta

Parametro obrigatório

Padrão: None

is_month()

Esta tag condicional testa para ver se a página é uma página de arquivo baseada em meses

<?php is_month(); ?>

Parâmetros

Esta tag não requer parâmetro

is_page()

Esta tag condicional verifica se a página está sendo exibida

<?php is_page($page); ?>

Parâmetros

$page -> O ID da página ou o título da página ou o slug da página

Parametro opicional

Padrão: None

is_page_template()

Esta tag condicional verifica se um modelo de página está sendo usado em alguma página

<?php is_page_template($template); ?>

Parâmetros

$template -> O nome do arquivo modelo

Parametro opicional

Padrão: None

is_paged()

Esta tag condicional verifica se a página que está sendo exibida é paginada

<?php is_paged(); ?>

Parâmetros

Esta tag não requer parâmetro

is_plugin_active()

Esta tag condicional verifica se o plugin está ativo

<?php is_plugin_active($plugin); ?>

Parâmetros

$plugin -> O nome do plugin em sub-diretorio/file

Parametro obrigatório

Padrão: None

is_plugin_page()

Esta tag condicional verifica se quando a solicitação atual é um plugin da página do painel do adminstração

<?php is_plugin_page(); ?>

Parâmetros

Esta tag não requer parâmetro

is_search()

Esta tag condicional verifica se a página de resultado da busca está sendo exibida

<?php is_search(); ?>

Parâmetros

Esta tag não requer parâmetro

is_single()

Esta tag condicional verifica se a página de single post está sendo exibida

<?php is_single($post); ?>

Parâmetros

$post -> O ID do post ou o título do post ou o slug do post

Parametro opicional

Padrão: None

is_singular()

Esta tag condicional verifica se a página singular está sendo exibida

<?php is_singular($post_types); ?>

Parâmetros

$post_types -> Tipo do post para verificar a consulta

Parametro opicional

Padrão: None

is_sticky()

Esta tag condicional verifica se o post é tipo Sticky

<?php is_sticky($post_ID); ?>

Parâmetros

$post_ID -> O ID do post

Parametro opicional

Padrão: None

is_tag()

Esta tag condicional verifica se a pagina de arquivo tag está sendo exibida

<?php is_tag($slug); ?>

Parâmetros

$slug -> O slug da Tag

Parametro opicional

Padrão: None

is_taxonomy_hierarchical()

Esta tag condicional verifica se a taxonomia é hierárquica

<?php is_taxonomy_hierarchical($taxonomy); ?>

Parâmetros

$taxonomy -> O nome da taxonomia

Parametro obrigatório

Padrão: None

in_category()

Esta tag condicional verifica se o post atual ou qualquer outro post especificado é atribuido a qualquer das categorias estabelecidas

<?php in_category($category, $_post); ?>

Parâmetros

$category -> Categoria(s) estabelecida(s), pode ser especificado por ID, Nome ou Slug da categoria

Parametro obrigatório

Padrão: None

$_post -> O(s) post(s), é especificado pelo ID

Parametro opcional

Padrão: None

in_the_loop()

Esta tag condicional verifica se o loop está ativo no momento

<?php in_the_loop(); ?>

Parâmetros

Esta tag não requer parâmetro

is_active_sidebar()

Esta tag condicional verifica se a sidebar está ativa

<?php is_active_sidebar($index); ?>

Parâmetros

$index -> O ID da sidebar ou o nome da sidebar ou o número da sidebar

Parametro obrigatório

Padrão: None

is_dynamic_sidebar()

Esta tag condicional verifica se a sidebar atual tem widgets ativo

<?php is_dynamic_sidebar(); ?>

Parâmetros

Esta tag não requer parâmetro

is_user_logged_in()

Esta tag condicional verifica se visitante atual está logado

<?php is_user_logged_in(); ?>

Parâmetros

Esta tag não requer parâmetro

has_excerpt()

Esta tag condicional verifica se o post atual contém Excerpt (resumo)

<?php has_excerpt($id); ?>

Parâmetros

$id -> O ID do post

Parametro opicional

Padrão: ID do post atual (exibido)

has_post_thumbnail()

Esta tag condicional verifica se o post atual contém post thumbnail (miniatura do post)

<?php has_post_thumbnail($post_id); ?>

Parâmetros

$post_id -> O ID do post

Parametro opicional

Padrão: ID do post atual (exibido)

pings_open()

Esta tag condicional verifica se o post atual são permitidos pings

<?php pings_open($post_id); ?>

Parâmetros

$post_id -> O ID do post

Parametro obrigatório

Padrão: None

email_exists()

Esta tag condicional verifica se o endereço de e-mail já foi registrado por algum usuário. Retorna o ID do usuário, se falso não existe nenhum.

<?php email_exists($email); ?>

Parâmetros

$email -> E-mail para ser verificado

Parametro obrigatório

Padrão: None

post_type_exists()

Esta tag condicional verifica se existe o tipo de post.

<?php post_type_exists($post_type); ?>

Parâmetros

$post_type -> Tipo de post

Parametro obrigatório

Padrão: None

taxonomy_exists()

Esta tag condicional verifica se existe a taxonomia existe.

<?php taxonomy_exists($taxonomy); ?>

Parâmetros

$taxonomy -> Nome da taxonomia

Parametro obrigatório

Padrão: None

username_exists()

Esta tag condicional verifica se o username já foi registrado por outro usuário. Retorna o ID do usuário, se falso não existe nenhum.

<?php username_exists($username); ?>

Parâmetros

$username -> Username para ser consultado

Parametro obrigatório

Padrão: None
